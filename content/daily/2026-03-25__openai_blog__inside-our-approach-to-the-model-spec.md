OpenAI가 Model Spec이라는 공개 프레임워크를 통해 AI 모델의 행동 기준을 제시합니다. 이 프레임워크는 AI 시스템이 발전함에 따라 안전성, 사용자 자유도, 책임성 간의 균형을 맞추는 것을 목표로 합니다. 모델 행동에 대한 투명하고 공개적인 가이드라인을 제공합니다.

</div>
<div class="lang-en" style="display:none">

OpenAI presents Model Spec as a public framework for AI model behavior standards. This framework aims to balance safety, user freedom, and accountability as AI systems advance. It provides transparent and open guidelines for model behavior expectations.
