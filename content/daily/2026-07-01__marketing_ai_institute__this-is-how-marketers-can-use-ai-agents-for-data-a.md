<div class="lang-ko">

SmarterX의 최근 프로젝트에 따르면 OpenAI Codex, Anthropic Claude Code 같은 AI 코딩 도구가 마케터의 복잡한 데이터 분석 업무에도 효과적으로 활용될 수 있다. 이 도구들은 기존에 개발자 전용으로 여겨졌으나, 지저분한 데이터를 정리하고 인사이트를 도출하는 마케팅 업무에 재목적화(repurpose)가 가능하다. Marketing AI Institute는 이 접근법이 마케터의 반복적이고 시간 소모적인 데이터 작업을 크게 줄여줄 수 있다고 강조한다.

</div>
<div class="lang-en" style="display:none">

A recent project at SmarterX demonstrates that AI coding tools like OpenAI's Codex and Anthropic's Claude Code can be repurposed beyond software development for common marketing data analysis tasks. These tools are effective at making sense of messy, unstructured data—one of the most tedious challenges marketers face. The Marketing AI Institute highlights this as a practical, accessible use case for AI agents in marketing workflows.

</div>

## Key Points

- OpenAI Codex·Anthropic Claude Code가 마케팅 데이터 분석에 재활용 가능함을 SmarterX 프로젝트로 실증
- 비개발자인 마케터도 AI 에이전트로 복잡한 데이터 정제·분석 작업 수행 가능
- 마케팅에서 가장 흔하고 반복적인 작업인 '지저분한 데이터 처리'에 즉각 적용 가능
- 개발자 도구와 마케팅 도구 간 경계가 AI 에이전트로 인해 빠르게 허물어지는 추세
