<div class="lang-en" style="display:none">

Vercel's Chief of Software Andrew Qu shared how the company built its agent framework called 'eve' and articulated why agents represent a new paradigm in software development. Key concepts emerging in agentic software include skills, sandboxes, and agent-readable websites. Vercel positions agents as a fundamentally different category of software rather than an incremental evolution.

- Vercel built an agent framework called 'eve' with a new architectural philosophy
- Agent-readable websites are emerging as a key design requirement for the agentic web
- Skills and sandboxes are foundational components of the new agentic software paradigm
- Agents are characterized as a qualitatively new kind of software, not just an automation layer
